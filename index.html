Wotstat - WN8 статистика за сессию

Мод статистики за сессию для World of Tanks

Download as .zip Download as .tar.gz View on GitHub

Возможности

  • Отображение WN8, WN6, EFF и промежуточных переменных из формулы расчета WN8. Например, переменных rDAMAGE, rSPOT, rFRAG, rDEF и rWIN.
  • Средний уровень танков, средний уровень боёв, опыт и кредиты заработанные за день.
  • Загрузка формата выводимого сообщения из файла шаблона с возможностью подстановки пользовательских макросов.
  • Отображение статистики за каждый бой. Макросы те же самые. Например, можно выводить рейтинг WN8, чистый доход или нанесенный урон за каждый бой в центр уведомлений,
  • Для всех макросов значений могут быть определены макросы цвета. Значения цветов берутся из файла конфига. Можно использовать градиентные цвета, или палитру.
  • Включение/отключение ежедневного обнуления статистики. Возможность выбора времени сброса.
  • Возможность ручного сброса статистики.
  • Вывод сообщения о результате недосмотренного боя в чат текущего боя.
  • Скрывание сообщений в системном канале по заданному шаблону.
  • Возможность отключить ведение статистики по некоторым типам боёв. (параметр ignoreBattleType в конфиге. номера типов боёв)
  • Раздельная статистика по танкам

Компиляция

Для компиляции этой модификации нужен Python 2.7.

python2.7 compiler.py
